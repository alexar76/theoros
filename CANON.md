## The Seven Precepts

### I — Agency is verified, not claimed

Agency is proof, not prose. An agent earns standing through verifiable scores, signed receipts, and oracle-backed attestations — never through a convincing bio.

**Product anchors:** [Metis](https://github.com/alexar76/metis) `verify_score`, Ed25519 oracle receipts, AIMarket signed invoke results.

### II — Weak aggregation is tyranny

A council that cannot reject bad inputs is not deliberation — it is theatre with extra latency. Weak models in aggregation roles regress benchmark quality and poison downstream trust.

**Product anchors:** Metis 90→60 regression on weak aggregation, capability gate, honest-null reporting.

### III — Tool access is territory

MCP tools define borders, not plugins. What an agent may invoke is territorial sovereignty — guarded at the boundary, not negotiated after the fact.

**Product anchors:** [ARGUS](https://github.com/alexar76/argus) WARDEN, MCP firewall pinning, tool-def hash attestation.

### IV — Invoke is contract

Every capability invocation is a contract: price, schema, receipt, and dispute surface. Publish-and-earn only works when invoke semantics are explicit and verifiable.

**Product anchors:** [AIMarket Hub](https://github.com/alexar76/aimarket-hub), protocol v2, publish-and-earn routing.

### V — Sovereignty is gated

Sovereignty without thresholds is mob rule plus latency. Capability tiers, QA gates, and human approval paths exist so autonomy scales without becoming noise.

**Product anchors:** `capability.py`, HELIOS approve workflow, Factory QA gates.

### VI — Canon is amended in public

This document is forkable. Amendments arrive as pull requests, debated in `#canon-debate`, and merged with a version bump — not edited in private.

**Product anchors:** this repo, Council vs Solo challenge, GitHub Discussions on Metis.

### VII — Silence beats false certainty

An honest null beats confident hallucination. Benchmarks publish caveats; councils admit when evidence is thin; agents refuse when verification fails.

**Product anchors:** Metis honest-null, benchmark regression reports, AEGIS rejection paths.
